1. Open Liberty 개요​

Open Liberty는 온프레미스 환경 뿐만 아니라 클라우드나 마이크로 서비스 아키텍처 환경의 더 빠른 애플리케이션의 개발 및 전달을 위해 경량화된 WAS 입니다. 해당 프로파일은 기본적으로 Java 21.0/Java EE 8/Jakarta EE 10 표준을 지원하면서도 적은 용량에, 빠르게 구동이 가능한 경량화 WAS 로 개발자들의 개발 생산성을 향상시킵니다.

2. 전통적인 WebSphere(WAS) vs Open Liberty​

Open Liberty 는 컨테이너 환경 뿐만 아니라 마이크로서비스 아키텍처를 위해 최적화된 애플리케이션 서버입니다.​​​

모듈별 구성 경량화 64MB Footprint 최적화 마이크로서비스
컨테이너 환경
컨테이너 환경 및 마이크로서비스를 위한 Open Liberty​
  • - 대단위 개발자를 위한 Java Framework (Java EE 기반의 Spring 포함)​
  • - Open Liberty 는 경량의 구성가능한 런타임 아키텍처로 디자인됨​
  • - Open Liberty 아키텍처 + Java EE8 / Jakarta EE10 = 마이크로서비스를 위한 입증된 엔터프라이즈 개발 플랫폼​​
  • - DevOps 워크플로우와 Docker 컨테이너 전략에 손쉽게 통합 가능​​
  • - Cloud 기반의 데이터와 인지 서비스에 완벽하게 연결​

Open Liberty 는 빠른 확장 및 서비스의 신속성을 위해 최적화된 구조를 가진 새로운 WAS 인프라이며,
FNG Soft 에서는 온프라미스 환경에서의 통합관리를 위해 엔터프라이즈급 관리도구를 제공합니다.​​

런타임과 설정, 애플리케이션을 압축형태로 패키지 가능​
중앙 관리 노드에서 서버 / 클러스터 관리 가능​​
대상 서버로 ftp 전송 후에 unzip 만으로 바로 수행 가능​
